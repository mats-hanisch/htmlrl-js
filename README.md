# htmlrl (HTML Render Language)

A simple yet powerful HTML template rendering engine for Node.js.

This library is **not** JavaScript-based like EJS or Pug.  
Instead, it introduces its **own domain-specific rendering language**, designed for simplicity, safety and explicitness.

---

## Features

- Simple HTML template syntax
- Automatic HTML escaping
- Immutable render data
- Fail-fast error handling
- Bytecode compilation + interpreter
- Cache-based compilation
- TypeScript support

---

npm install htmlrl
